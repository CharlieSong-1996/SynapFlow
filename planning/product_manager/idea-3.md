# SynapFlow 需求方案 - 数据节点分类与可视化节点（方案1）

## 1. 背景与目标
在既有“数据节点 + 处理器节点”的核心模型下，进一步细分数据节点职责，并明确可视化能力的最小实现路径。目标是：
- 保持节点心智简单。
- 让输出数据可在画布中被直观展示。
- 避免引入复杂看板、图层与炫技交互，优先满足直接可读。

## 2. 本次确认的关键决策
1. 采用可视化方案1：可视化作为数据节点的一种输出形态。
2. 渲染代码受框架强制约束，不是自由分散写法：
   - 可视化输出必须符合受限格式（如 SVG 或 HTML 子集）。
   - 由框架统一解析与渲染，不引入额外节点运行时。
3. 不追求复杂动效与重交互看板，首版仅做最直观的数据呈现。
4. 过时输出节点（Stale Output）画布规则：
   - 默认以 Collapsed 形式展示。
   - 与关联 Processor 保持虚线连接。
   - 用户可点击展开查看详情。
   - 若该过时输出节点已连接下游 Processor，则不执行折叠，仅将相关连线由实线改为虚线。

## 3. 数据节点分类定义（V1）
### 3.1 自由数据节点（Free Data Node）
- 创建来源：用户手动创建，或由已有数据节点复制而来。
- 权限特性：可独立编辑、复制、删除（受项目权限约束）。
- 流转特性：仅提供输出，可作为 Processor 输入。

### 3.2 输出型数据节点（Processor-bound Output Data Node）
- 创建来源：仅由 Processor 节点创建，用户不可直接新建。
- 绑定关系：与创建它的 Processor 强绑定。
- 生命周期：
  - Processor 重定义输出 Meta 后，旧输出节点退化为 Stale Output。
  - Stale Output 保留关联关系，用于追溯和对比。
- 流转特性：可作为下游 Processor 输入。

### 3.3 可视化输出数据节点（Visual Output as Data Node）
- 定位：仍属于数据节点体系，不新增第三类基础节点。
- 识别方式：在输出节点 Meta 中设置可视化标识与受限渲染格式。
- 渲染机制：画布渲染器识别标识后，直接在节点卡片中渲染。
- 首版范围：静态或轻交互展示，强调“直观可读”。

## 4. 交互流程（最小可用）
1. 用户连接上游数据节点到 Processor。
2. Processor 在 Meta 中声明输出为普通数据或可视化数据。
3. 画布生成输出节点：
   - 普通输出：标准数据卡片。
   - 可视化输出：节点内直接渲染 SVG/HTML 结果。
4. Processor 输出定义更新时：
   - 旧输出标记为 Stale Output。
   - 若无下游依赖：节点折叠 + 虚线关系。
   - 若有下游依赖：节点保持展开，仅将连线改为虚线。

## 5. 规则矩阵（V1）
| 节点类型 | 用户直接创建 | 可独立编辑 | 可独立删除 | 可作为Processor输入 | 可被渲染器直接渲染 |
|---|---|---|---|---|---|
| 自由数据节点 | 是 | 是 | 是 | 是 | 否 |
| 输出型数据节点 | 否 | 否（随Processor定义） | 否（受绑定约束） | 是 | 否 |
| 可视化输出数据节点 | 否（由Processor产出） | 否（随Processor定义） | 否（受绑定约束） | 视为数据输入（可配置） | 是 |

## 6. 可用性与一致性建议
1. 统一图例：虚线仅表达“过时关系”，避免复用为其他语义。
2. Collapsed 的可发现性：在节点标题显示 Stale 标记与版本号。
3. 展开行为保持一致：点击节点头部展开，二次点击折叠。
4. 可视化节点提供“查看原始数据”入口，避免只见图不见数。

## 7. 验收标准（MVP）
1. Processor 输出 Meta 可声明可视化格式，并被画布自动识别渲染。
2. 渲染格式不合规时，节点展示可读错误而非空白区域。
3. Processor 重定义输出后，旧输出正确标记为 Stale 并应用虚线规则。
4. 当 Stale 输出有下游依赖时，不可自动折叠，仅连线虚线化。
5. 全流程不引入新的“可视化处理器节点”类型。

## 8. 暂不纳入范围（Out of Scope）
- 多图层看板与复杂联动。
- 高级动画与可视化特效。
- 独立可视化节点运行时与专属 Agent。

## 9. 后续演进点
1. 增加可视化模板库（仍基于受限 SVG/HTML 规范）。
2. 增加从可视化节点回跳数据明细的双向定位。
3. 在不改变节点模型前提下，补充最少量交互控件（筛选、排序）。
