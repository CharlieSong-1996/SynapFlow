SynapFlow
=========

SynapFlow is a highly customizable, Agent-driven data pipeline canvas. 
It bridges visual data flows with LLM-powered programming by separating 
Data Nodes (e.g., SQL outputs) from Processor Nodes. Each Processor Node 
is powered by an independent AI Agent with its own memory and chat interface. 
Instead of writing code manually, users simply chat with the Agent to
automatically generate, modify, and execute processing scripts in various 
languages, dynamically transforming inputs into new data nodes.

SynapFlow 是一个高度定制化的“AI + 数据流”编排平台。它巧妙地将数据节点
（如查询结果集）与处理器节点分离开来。每个处理器节点都绑定了一个拥有独
立记忆的 AI Agent。用户无需手动编写代码，只需在画布上连接数据，并与节
点的专属 Agent 对话，即可动态生成和修改底层执行脚本。Agent 会自动处理
输入数据，并衍生出全新的输出数据节点。
